    Wind Tunnel Controller

    • Tesla
    • Date : December 2016
    • Used Technologies : Rapid Prototyping, LabVIEW Control System Design, Multi Motor/Sensor Interfacing, Signal Testing, Data Analysis

    Since Tesla's products generally prioritize NDA, it is a challenge to test the aerdynamacy of Tesla vehicles. Originally to perform these tests, Tesla would be required to pay wind tunnel facilities for their space, Engineers, and hardware. This risked possible leakage of sensitive information on new vehicles. As such, the goal was to design a test setup to measure the aerodynamacy of Tesla vehicles that Tesla Engineers could perform without risking leakage of propietary information. This involved learning how aerodynamic testing is done, determining the apprioriate hardware required, putting together a test setup, and ensuring it was simple enough that Engineers without much experience could use it.

    • ● Programmed LabVIEW control system with GUI
    • ● Interfaced linear actuators for (a)synchronous ride height control and real-time road simulation
    • ● Interfaced with multiple sensors, utilizing sensor fusion
    • ● Created 3D visual to see dynamic pressure changes over chassis of vehicle
    • ● Implemented I/O data logging with dynamic data referencing for reverse video playback

    A-Flex Offline

    • Magna
    • Date : April 2015
    • Used Technologies : Siemens NX Product Design, Mechanical Analysis, G Code Scripting, Simulations

    The goal was to redesign an end effector to improve the efficiency of the weld line. Due to the long process of completing such a task, involving "design a prototype, have it verified, manufacture the component, test it, and redo to further improve it", the team believed it to take 6 months to complete. I managed to complete it within 4 months by utilizing simulation software. I took initiative to scower the web for software that would allow me to test various prototypes on an offline simulation of the weld line. Once the best prototype was found, it was implemented into the weld line.

    • ● Designed prototypes on Siemens NX, optimized using FEA
    • ● Scripted G-Code to improve weld path for newly designed prototype
    • ● Facilitated testing via RoboGuide Simulation, minimizing lead time
    • ● Improved cycle time by 140% -> $160,000 savings per year
    • ● Presented to GM, propogating idea to simulate other lines
